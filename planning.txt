 Purpose:
       Create a single-page React + TypeScript app that fetches a random fun fact from the Useless Facts API and displays it to the user. The user can click a button to fetch a new fact.

 Tech Stack:
        React with TypeScript
        SCSS for styling with:
                -Variables
                -Mixins
                -Nesting
                -Responsive design
       API: https://uselessfacts.jsph.pl/random.json?language=en

 Features: 
       Fetch a random fact on first load
       Display fact content and source
       Button to fetch a new fact
       Clean component structure
       SCSS module for styling

Optional: 
       Add mixins/variables if needed for future extension
